# linux-security-suite

Blue team security tooling for Linux servers. Each tool is self-contained and built to run on a live system.

## Tools

| Tool | Description | Language |
|------|-------------|----------|
| [port-scan-detector](./port-scan-detector/) | Detects incoming port scans in real time | Go |

## Philosophy

Server-side, defensive, minimal dependencies. Tools are designed to be auditable — small enough to read and understand completely.
